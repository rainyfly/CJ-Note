2. Engine Python API

engine.py 提供了 Python 嵌入式引擎接口，无需启动 HTTP 服务：

Engine 使用方式

import sglang as sgl

engine = sgl.Engine(model_path="meta-llama/Llama-3-8B-Instruct")

# 同步生成
output = engine.generate(
    prompt="Hello, how are you?",
    sampling_params={"temperature": 0.8, "max_new_tokens": 256}
)

# 异步生成
async for chunk in engine.async_generate(...):
    print(chunk)

engine.shutdown()

Engine 类的核心方法：

方法说明
generate()同步文本生成
async_generate()异步文本生成 (支持流式)
encode()文本编码为 embedding
classify()文本分类
update_weights_from_disk()从磁盘热更新权重
update_weights_from_tensor()从 Tensor 热更新权重
load_lora_adapter()加载 LoRA 适配器
shutdown()关闭引擎

5. Anthropic 兼容

entrypoints/anthropic/ 实现了 Anthropic Messages API (/v1/messages) 兼容，支持 Anthropic SDK 直接接入。

6. Ollama 兼容

entrypoints/ollama/ 实现了 Ollama API 兼容，支持 /api/generate 和 /api/chat 端点，方便从 Ollama 迁移。

7. gRPC Server

grpc_server.py 和 srt/grpc/ 提供了 gRPC 服务接口，使用 protobuf 定义的 sglang_scheduler.proto，适用于高性能 RPC 场景。

8. SGLang 原生 API

SGLang 原生 /generate 端点支持更多高级功能：

  • regex - 正则约束解码
  • json_schema - JSON Schema 约束
  • return_logprob - 返回 logprob
  • session_id - 会话管理
  • input_ids - 直接传入 token IDs
  • lora_path - 指定 LoRA 适配器

9. Warmup 预热

warmup.py 在服务启动后执行预热请求，触发 CUDA Graph 捕获和 JIT 编译，确保首次请求的延迟正常。
